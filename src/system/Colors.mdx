# Colors

## Luminance grades

| Grade         | Min   | Max   |
|---------------|-------|-------|
| 0 (white)     | 1     | 1     |
| 5             | 0.85  | 0.95  |
| 10            | 0.75  | 0.84  |
| 20            | 0.64  | 0.74  |
| 30            | 0.381 | 0.624 |
| 40            | 0.288 | 0.3   |
| 50            | 0.175 | 0.183 |
| 60            | 0.1   | 0.125 |
| 70            | 0.05  | 0.093 |
| 80            | 0.026 | 0.045 |
| 90            | 0.01  | 0.025 |
| 100 (black)   | 0     | 0     |

Choosing accessible color combinations:

* Color combinations of **40+** result in a 3:1 contrast ratio
* Color combinations of **50+** result in a 4.5:1 contrast ratio
* Color combinations of **70+** result in a 7:1 contrast ratio
